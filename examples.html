Here is python code that can be used to make a pseudo calculator.

  number1 = input("type a number: ")
  number2 = input("type a second number: ")
  print("here the directions + is addition - is subtraction / is dividing * is multiply")
  operation = input("what do yo want to do? ")

  if operation == "+":
    result = int(number1) + int(number2)
  elif operation == "-":
    result = int(number1) - int(number2)
  elif operation == "/":
    result = int(number1) / int(number2)
  elif operation == "*":
    result = int(number1) * int(number2)

  print(result)
